# Super-Store-Sales-Analysis ( Power-BI )
## Project Objective 
Based on the 4-year dataset to understand the sale market and grow more sales in next years.

## Dataset Used
- <a href="https://github.com/myolwinhein-data/Super-Store-Sales-Analysis-Power-BI-/blob/main/SuperStore.csv">superstore data</a>

## Questions ( KPIs )
- Which products stand as top five over revenue?
- Comparing revenue and profit quarterly.
- Who are our top customers?
- Which state provide the most revenue?

## Process
- Apply the data transforming and data cleaning and correct data formats.
- Create star schema for the relationship and to get better dashboard.
- Use dax functions to calculate total sales and profits.
- Apply slicers and charts to get meanful insights.
